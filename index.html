Forest Fire Risk Analysis

Developed a forest fire susceptibility map using GIS-based Multi-Criteria Decision Analysis (MCDA).

Data Used:
DEM, DL_FIRE_SV-C2 fire data, WorldCover LULC, CHIRPS rainfall, OSM rivers, roads and settlements.

Methods:
Raster analysis, reclassification, weighted overlay, and spatial analysis using QGIS.

Result:
Forest fire susceptibility zones classified into low, medium, and high-risk categories.

Flood Susceptibility Analysis

Developed a Flood Susceptibility Map for Jabalpur District using GIS-based Multi-Criteria Decision Analysis (MCDA). The analysis identifies areas vulnerable to flooding by integrating multiple environmental and terrain factors.

Data Used:
Digital Elevation Model (DEM), Land Use/Land Cover (LULC), rainfall data, drainage network, and district boundary.

Methodology:
The workflow included spatial data preprocessing, clipping by mask layer, raster reclassification, weighted overlay analysis, raster calculator, and classification of flood susceptibility zones.

Tools:
QGIS, Raster Analysis, Multi-Criteria Decision Analysis (MCDA)

Result:
Generated a flood susceptibility map classifying the study area into Very Low, Low, Moderate, High, and Very High susceptibility zones. The output supports disaster management, land-use planning, and infrastructure decision-making.
