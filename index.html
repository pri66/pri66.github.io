Forest Fire Risk Analysis

Developed a forest fire susceptibility map using GIS-based Multi-Criteria Decision Analysis (MCDA).

Data Used:
DEM, DL_FIRE_SV-C2 fire data, WorldCover LULC, CHIRPS rainfall, OSM rivers, roads and settlements.

Methods:
Raster analysis, reclassification, weighted overlay, and spatial analysis using QGIS.

Result:
Forest fire susceptibility zones classified into low, medium, and high-risk categories.

Flood Susceptibility Analysis

Developed a Flood Susceptibility Map for Jabalpur District using GIS-based Multi-Criteria Decision Analysis (MCDA).

Data Used:
DEM, LULC, rainfall data, drainage network, and district boundary.

Methodology:
Spatial preprocessing, raster reclassification, Raster Calculator, and Weighted Overlay Analysis.

Tools:
QGIS, Raster Analysis, MCDA.

Result:
Generated flood susceptibility zones from Very Low to Very High risk categories.

Watershed Delineation and Hydrological Analysis

Developed a watershed delineation workflow using DEM data to analyse terrain characteristics and drainage patterns.

Data Used:
Digital Elevation Model (DEM), watershed boundary data, and elevation-derived hydrological parameters.

Methods:
DEM preprocessing, fill sinks, flow direction, flow accumulation, stream extraction, and watershed delineation using QGIS.

Tools:
QGIS Raster Analysis and Hydrological Modelling.

Result:
Generated watershed maps showing drainage networks and terrain characteristics.
